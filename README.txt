This is my first javaScript app.

Project's Title =============
The Name of this project is: Dad Joke generator....
Project Description =============
This project will supply you with a random dad joke until you've had enough. 
Set the joke limit and and the Dad Joke generator will do the hard-ish work for you. 
There are currently 138 dad jokes in total.
It will supply a table with every joke in the program upon completion of the program.
